# Attack on Titan / A.O.T. Wings of Freedom

- Fix resolution above 1920x1080
- Fix aspect ratio for ultrawide monitors(HUD will stretch)

## Installation
1. Download the [SUWSF zip](https://github.com/santerus349/SUWSF-Patches/releases/download/AOT/Attack_on_Titan_SUWSF.zip).
2. Extract contents of the zip into the game folder.

### Steam Deck/Linux Additional Instructions
🚩**You do not need to do this if you are using Windows!**  
- Open up the game properties in your launcher and add `WINEDLLOVERRIDES="version=n,b" %command%` to the launch options.

## Configuration
- Edit `SUWSF.ini` in the game folder to disable/enable patches.
